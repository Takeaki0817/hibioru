# プロダクト概要

**サービス名**: ヒビオル (hibioru)
**コンセプト**: 「日々を織る」 - 1日の中での心の動きを記録する思考記録アプリ

Twitter的な投稿感覚で、日々の瞬間をログとして残す。

**目的**:
- 考えていることを言語化して記録する習慣を作る
- 誰にも見せない、雑に吐き出せる場所を作る
- 記録をAIも使って振り返りやすくする

---

## コア機能

### 1. ストリーク（継続記録）システム
Duolingo式の継続モチベーション設計
- 毎日0:00リセット
- 1件以上の記録で継続カウント
- 「失いたくない資産」としての心理効果

### 2. ほつれ（スキップ機能）
継続を守るためのセーフティネット
- 週2回付与、繰り越しなし
- **自動消費**: 記録がない日に自動で使用
- 完全な失敗を防ぐ「途切れても大したことない」感

### 3. 最小単位入力
「やらない理由」を潰す設計
- 絵文字1つでもOK
- 2タップで記録完了
- 画像添付可能（1投稿1枚）

### 4. プッシュ通知
ADHD当事者の「忘れる」「自分から動けない」への対策
- 定時リマインド + 追いリマインド
- 記録するまで最大3回通知

## こんな悩みを持つ人へ

- 気がついたら1日、1週間、1年が過ぎて、何をしていたかあまり覚えていない
- 日記や日報を続けようとしたけど、続いたことがない
- 人に話せないことや、読んでもらうための文章にするのが面倒くさい
- 頭の中のモヤモヤを書き出してスッキリしたい
- 愚痴や怒り、悲しみを吐き出す場所がほしい
- 考え事で集中できない、眠れないときに、頭を空っぽにしたい
- 記録したデータをAIを使って振り返りたい

---

## 価値提案

### Duolingoから学んだ設計思想

1. **損失回避** - ストリークが「失いたくない資産」になる
2. **最小単位が極端に小さい** - 「やらない理由」が潰される
3. **セーフティネット** - ほつれで完全な失敗を防ぐ
4. **可視化** - 記録が目に見える形で残る

### 差別化ポイント

- 「瞬間記録 x 継続支援」の組み合わせ
- Duolingo式ゲーミフィケーション
- ADHD当事者が設計（当事者視点）

## 用語定義

| 用語 | 意味 |
|------|------|
| **ヒビオル** | サービス名。アニメ映画『さよならの朝に約束の花をかざろう』に登場する布。縦糸が「月日」、横糸が「人のなりわい」を表し、日々の出来事や人生そのものを意味する |
| **ほつれ** | 継続を守るためのスキップ機能 |
| **ほつれ直し** | ほつれを使って途切れを防ぐこと |

## 設計原則

### やってはいけないこと（ソーシャル機能）
- SNS化しない（無限スクロール、いいね数表示、おすすめなし）
- 比較・競争を煽らない
- ADHDの報酬系を余分に刺激しない

### 守るべきこと
- 報酬は「継続」に対してのみ
- 無料で十分使えることを維持（課金圧を強くしない）
- ADHD当事者にストレスを与えない設計を優先
