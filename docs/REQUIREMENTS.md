# ヒビオル 要件定義書

## 概要

**サービス名**: ヒビオル（hibioru）
**コンセプト**: 「日々を織る」- 1日の中での心の動きを記録する思考記録アプリ

Twitter的な投稿感覚で、日々の瞬間をログとして残す。

**目的**:

- 考えていることを言語化して記録する習慣を作る
- 誰にも見せない、雑に吐き出せる場所を作る
- 記録をAIも使って振り返りやすくする

---

## 用語定義

| 用語           | 意味                                                                                                                                                       |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ヒビオル**   | サービス名。アニメ映画『さよならの朝に約束の花をかざろう』に登場する布。縦糸が「月日」、横糸が「人のなりわい」を表し、日々の出来事や人生そのものを意味する |
| **ほつれ**     | 継続を守るためのスキップ機能                                                                                                                               |
| **ほつれ直し** | ほつれを使って途切れを防ぐこと                                                                                                                             |

---

## こんな悩みを持つ人へ

- 気がついたら1日、1週間、1年が過ぎて、何をしていたかあまり覚えていない
- 日記や日報を続けようとしたけど、続いたことがない
- 人に話せないことや、読んでもらうための文章にするのが面倒くさい
- 頭の中のモヤモヤを書き出してスッキリしたい
- 愚痴や怒り、悲しみを吐き出す場所がほしい
- 考え事で集中できない、眠れないときに、頭を空っぽにしたい
- 記録したデータをAIを使って振り返りたい

---

## 解決する課題

| 課題                     | 対策                                     |
| ------------------------ | ---------------------------------------- |
| メンタル落ちると書けない | プッシュ通知 + 最小入力（絵文字1つでOK） |
| 忘れる                   | 定時リマインド                           |
| 継続の意味を見失う       | 継続記録表示、AI連携での振り返り         |
| 1ヶ月で新鮮さが消える    | ゲーミフィケーション（継続記録、ほつれ） |
| スマホで操作しづらい     | PWA、2タップで記録完了                   |
| デザインが堅い           | 軽い見た目、遊び心のある用語             |

### 主要な仕様概要

| 項目           | 仕様                                                |
| -------------- | --------------------------------------------------- |
| 1日の投稿数    | 最大20件                                            |
| 画像枚数       | 1投稿最大2枚、1日最大5枚                            |
| 画像サイズ     | 圧縮後200KB以下                                     |
| 編集可能期間   | 24時間以内                                          |
| 削除           | 論理削除                                            |
| 下書き         | ローカルストレージに自動保存                        |
| 日付切り替え   | 毎日0:00 (JST)                                      |
| ほつれ付与     | 週2回（月曜0:00にリセット、繰り越しなし）           |
| リマインド     | 最大5つの時刻を設定可能                             |
| 追いリマインド | 定時後60分、最大2回まで（記録すると自動キャンセル） |

## マネタイズ設計

### 方針

- Phase 1-3（MVP〜β）: 無料、制限なし
- Phase 4（ローンチ）: βテストのデータを元に制限値を決定し実装
- 目標: サーバー維持費（月$25〜）を賄う

### プラン構成（案）

**無料プラン**

| 項目   | 制限    |
| ------ | ------- |
| 投稿数 | 15件/日 |
| 画像   | 5枚/月  |

カジュアルユーザーには十分な制限。

**プレミアムプラン**

| プラン | 価格（税込） | 備考                       |
| ------ | ------------ | -------------------------- |
| 月額   | 480円        | -                          |
| 年額   | 4,200円      | 約27%お得（月350円換算）   |

| 項目   | 内容   |
| ------ | ------ |
| 投稿数 | 無制限 |
| 画像   | 無制限 |

**単発課金**

| アイテム            | 価格（税込） |
| ------------------- | ------------ |
| ほつれ追加（2回分） | 120円        |

### 決済方式

| 経路                | 手数料 | 価格（税込） |
| ------------------- | ------ | ------------ |
| PWA（Stripe直接）   | 4.3%   | 480円/月     |
| iOSアプリ（将来）   | 15%    | 580円/月     |

### 収益目標

- 最低目標: 月$25（約3,800円）= Supabase Proプラン維持費
- 達成条件: 月額ユーザー約9人 or 年額ユーザー約1人/月

### 価格設定根拠

| 項目             | 内容                                              |
| ---------------- | ------------------------------------------------- |
| 競合比較         | muute（1,650円/月）、Day One（3,800円/年）の1/3〜同等 |
| 手数料控除後実収入 | 月額459円、年額4,019円                            |
| 消費税           | 免税事業者（年間売上1,000万円以下）のため納付不要 |

### 注意事項

- 制限値はβテストのデータを元に最終決定
- 無料で十分使えることを維持する（課金圧を強くしない）
- ADHD当事者にストレスを与えない設計を優先
