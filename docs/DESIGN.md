# ヒビオル - デザインディレクション

## 調査

### 対象ユーザーの特性（ADHD当事者）

- 認知負荷が高いと離脱する
- 視覚的なノイズに敏感
- 「続ける」こと自体が最大の課題
- ネガティブなフィードバックでモチベーションが下がりやすい

### 競合アプリの傾向

- 多くの日記アプリは機能が多すぎる
- ゲーミフィケーション系は派手すぎることがある
- シンプルすぎると継続の動機付けが弱い

### Duolingoから学んだこと

- 視覚的に「成果が見える」ことが重要
- アニメーションは控えめだが達成感を演出
- 警告は脅かさず、行動を促す

---

## 結論

### デザイン方針

1. **視覚的に軽く** - 白/グリーン基調、余白を十分に
2. **情報を絞る** - 一画面に表示する要素を最小限に
3. **フィードバックは控えめに** - 派手なアニメーションより微細な動き
4. **ポジティブな表現** - 失敗を責めず、行動を促す文言

### カラー

- **プライマリ: グリーン系** - 成長・継続・健康のイメージ
- 警告は黄色、危険は赤だが、威圧的にならない彩度に抑える

### レイアウト

- ヘッダー/メイン/フッターの3層構造で統一
- 中央ボタンを目立たせ、主要アクションを促進
- カードは薄いボーダーで区切り、影は控えめに

### アニメーション

- タップ時のスケール変化（0.95程度）で反応を示す
- 警告状態は控えめな振動で注意を引く
- 過度な動きは認知負荷になるため避ける
