# ヒビオル - デザインディレクション

## 調査

### 対象ユーザーの特性（ADHD当事者）

- 認知負荷が高いと離脱する
- 視覚的なノイズに敏感
- 「続ける」こと自体が最大の課題
- ネガティブなフィードバックでモチベーションが下がりやすい

### 競合アプリの傾向

- 多くの日記アプリは機能が多すぎる
- ゲーミフィケーション系は派手すぎることがある
- シンプルすぎると継続の動機付けが弱い

### Duolingoから学んだこと

- 視覚的に「成果が見える」ことが重要
- アニメーションは控えめだが達成感を演出
- 警告は脅かさず、行動を促す

---

## 結論

### デザイン方針

1. **視覚的に軽く** - 白/グリーン基調、余白を十分に
2. **情報を絞る** - 一画面に表示する要素を最小限に
3. **フィードバックは控えめに** - 派手なアニメーションより微細な動き
4. **ポジティブな表現** - 失敗を責めず、行動を促す文言

### カラーシステム

OKLch色空間による知覚均等なカラーパレットを採用。

#### プライマリ: Sage Green（Hue: 145）

成長・継続・健康を象徴する落ち着いたグリーン。

| トークン | ライト | ダーク | 用途 |
|---------|--------|--------|------|
| primary-50 | L=0.97 | L=0.20 | 薄い背景 |
| primary-100 | L=0.93 | L=0.28 | ホバー状態 |
| primary-200 | L=0.85 | L=0.38 | アクティブ状態 |
| primary-300 | L=0.75 | L=0.50 | フォーカス |
| primary-400 | L=0.65 | L=0.60 | 軽い強調 |
| primary-500 | L=0.55 | L=0.70 | ベースカラー |
| primary-600 | L=0.45 | L=0.80 | 濃い強調 |
| primary-700 | L=0.35 | L=0.88 | 最も濃い |

#### アクセント: Warm Coral（Hue: 25）

達成感・報酬を表現するあたたかいコーラルトーン。

| トークン | 用途 |
|---------|------|
| accent-400 | 今日の日付、選択状態 |
| accent-500 | ボタンアクセント |

#### 報酬カラー: Golden（Hue: 85）

ストリーク達成など特別な瞬間に使用。

#### シャドウシステム

| トークン | 用途 |
|---------|------|
| shadow-xs | 微細な浮き |
| shadow-sm | カード、入力 |
| shadow-md | ホバー状態 |
| shadow-lg | モーダル |
| shadow-focus | フォーカスリング |

#### グラデーション

| トークン | 用途 |
|---------|------|
| gradient-subtle | 背景の微妙な深み |
| gradient-primary | ボタン（将来用） |
| gradient-reward | 達成演出 |

#### Glassmorphism

ダイアログ・シートのみに控えめに適用。
- `glass-bg`: 半透明背景
- `glass-blur`: 12pxブラー

#### アニメーション変数

| トークン | 値 | 用途 |
|---------|-----|------|
| duration-fast | 100ms | 即座の反応 |
| duration-normal | 200ms | 標準 |
| duration-slow | 300ms | ゆっくり |
| ease-default | ease-out | 標準イージング |
| ease-spring | spring | 弾力のある動き |

#### セマンティックトークン

- `--interactive-hover/active/focus`: インタラクション状態
- `--text-primary/secondary/tertiary`: テキスト階層
- `--success/error`: 状態カラー

### レイアウト

- ヘッダー/メイン/フッターの3層構造で統一
- 中央ボタンを目立たせ、主要アクションを促進
- カードは薄いボーダーで区切り、影は控えめに

### アニメーション

- タップ時のスケール変化（0.95程度）で反応を示す
- 警告状態は控えめな振動で注意を引く
- 過度な動きは認知負荷になるため避ける
