# ヒビオルへようこそ

ADHD当事者のための瞬間記録アプリ

<ReadTime minutes={2} />

---

## ヒビオルとは？

**ヒビオル**は、「日々」と「織る」を組み合わせた造語です。毎日の小さな記録を織り重ねていくように、あなたの思考や感情を気軽に記録できるアプリです。

<Tip type="success">
  ADHDの特性を考慮し、**最小限の操作**で記録できるよう設計されています。絵文字1つから投稿OK！
</Tip>
