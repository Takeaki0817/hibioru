# 用語説明

ヒビオルで使われる用語を解説します

<ReadTime minutes={3} />

---

<div className="grid gap-4 sm:grid-cols-2">

<GlossaryCard term="ヒビオル" emoji="🧵">
  **このアプリの名前**

  「日々」と「織る」を組み合わせた造語。毎日の小さな記録を織り重ねていくイメージ。糸を一本一本紡ぐように、日々の思考を記録します。
</GlossaryCard>

<GlossaryCard term="ほつれ" emoji="🛡️">
  **継続を守るスキップ機能**

  記録できない日を「守る」ための機能。週2回まで使用可能で、毎週月曜0:00にリセット。記録がない日に自動消費されます。
</GlossaryCard>

<GlossaryCard term="ストリーク" emoji="🔥">
  **連続記録日数**

  Duolingo式の継続モチベーション設計。毎日0:00（JST）にリセット判定。記録またはほつれで継続し、途切れると0にリセット。最長記録は保持されます。
</GlossaryCard>

<GlossaryCard term="タイムライン" emoji="📜">
  **記録の一覧画面**

  日付ごとに記録がグループ化されて表示。上下スクロールで過去の記録を閲覧でき、カレンダーからジャンプも可能です。
</GlossaryCard>

<GlossaryCard term="エントリ" emoji="✏️">
  **1つの記録**

  テキストや画像を含む1つの記録単位。絵文字1つから投稿可能で、1日に最大20件まで作成可能です（βテスト版）。
</GlossaryCard>

<GlossaryCard term="カレンダー" emoji="📅">
  **月間の記録概要**

  1ヶ月の記録状況を俯瞰できます。記録がある日は緑色、ほつれを使った日は点線で表示。日付をタップするとその日にジャンプします。
</GlossaryCard>

<GlossaryCard term="エクスポート" emoji="📤">
  **データの書き出し**

  記録をJSON/Markdown形式でダウンロード。バックアップや他ツールへの移行に便利。AIツールに読み込ませて分析することも可能です。
</GlossaryCard>

</div>
