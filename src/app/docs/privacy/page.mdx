## 3. 収集する個人情報

本サービスでは、サービス提供のために以下の個人情報を収集します。

### 3.1 認証情報（Googleアカウント経由）

Googleアカウントでのログイン時に、以下の情報を取得します。

- メールアドレス
- 表示名（Googleアカウントの名前）
- プロフィール画像URL

<Tip type="info">
Googleアカウントのパスワードは本サービスに送信されません。認証はGoogle OAuth 2.0を通じて安全に行われます。
</Tip>

### 3.2 ユーザー生成コンテンツ

- **エントリー（記録）**: テキスト内容、作成日時、更新日時
- **画像**: アップロードされた画像（WebP形式に変換、最大200KB）
- **公開設定**: 各エントリーの公開/非公開設定

<Tip type="warning">
アップロードされた画像からは、位置情報などのEXIFメタデータは自動的に除去されます。
</Tip>

### 3.3 利用状況データ

- **ストリーク情報**: 現在の継続日数、過去最高継続日数、最終記録日
- **ほつれ使用履歴**: 残り回数、使用日の履歴
- **通知設定**: リマインド時刻、タイムゾーン、通知の有効/無効設定

### 3.4 ソーシャル機能データ

- **フォロー関係**: フォロー中/フォロワーのユーザーID
- **達成イベント**: アチーブメントの種類、達成日時、達成値
- **お祝い**: 送受信したお祝いの記録
- **ソーシャル通知**: フォロー通知、お祝い通知の履歴

### 3.5 技術情報

- **プッシュ通知購読情報**: Web Pushエンドポイント、暗号化キー
- **ブラウザ情報**: ユーザーエージェント
- **アクセスログ**: IPアドレス、アクセス日時（サーバーログ）

## 6. 越境移転について

本サービスで収集した個人情報は、上記の第三者サービスを通じて、日本国外（主に米国）のサーバーに保存されます。

### 移転先の安全性確保措置

- **Supabase**: SOC 2 Type II認証、GDPR準拠、AWSインフラ使用
- **Vercel**: SOC 2 Type II認証、GDPR準拠
- **Google**: Privacy Shield後継の標準契約条項（SCC）に基づく移転

これらのサービス提供者は、個人情報保護法第28条に基づき、日本と同等の個人情報保護水準を確保するための措置を講じています。

## 9. ユーザーの権利

ユーザーは、個人情報保護法に基づき、以下の権利を有します。

### 9.1 開示請求

保有個人データの開示を請求することができます。

<Table
  headers={["項目", "内容"]}
  rows={[
    ["請求方法", "アプリ内フィードバック機能、またはメール"],
    ["手数料", "無料"],
    ["対応期間", "請求受領後30日以内"],
    ["提供形式", "JSON形式またはPDF形式"],
  ]}
/>

### 9.2 訂正請求

保有個人データに誤りがある場合、訂正を請求することができます。

- **対象**: 表示名、プロフィール画像
- **方法**: アカウント設定画面から直接変更可能

### 9.3 削除請求

保有個人データの削除を請求することができます。

- **方法**: アカウント設定画面の「アカウント削除」機能
- **効果**: 全データの完全削除（復元不可）
- **注意**: 削除後は同じGoogleアカウントで再登録しても、過去のデータは復元されません

### 9.4 利用停止請求

違法に取得された個人データ、または利用目的を超えて利用されている個人データについて、利用停止を請求することができます。

### 9.5 第三者提供停止請求

ソーシャル機能によるデータ共有を停止することができます。

- **方法**: アカウント設定画面から公開設定をオフに変更
- **効果**: フォロワーへの達成共有、エントリー共有が停止されます

## 11. 未成年者の利用

### 16歳未満のユーザー

16歳未満の方が本サービスを利用する場合は、保護者の同意が必要です。

- **同意の確認**: アカウント作成時に年齢確認と保護者同意の確認を行います
- **保護者の権利**: 保護者は、お子様のアカウントについて、開示・訂正・削除を請求することができます

### 保護者の方へ

お子様が本サービスを利用していることを発見し、同意していない場合は、アプリ内フィードバック機能またはメールでご連絡ください。該当アカウントのデータを削除いたします。
